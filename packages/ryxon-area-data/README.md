# Ryxon Area Data

中国省市区数据，适用于 Ryxon Area 组件。

## 安装

```shell
# 通过 npm
npm i @ryxon/area-data

# 通过 yarn
yarn add @ryxon/area-data

# 通过 pnpm
pnpm add @ryxon/area-data
```

## 使用

```ts
import { areaList } from '@ryxon/area-data';
```

## 数据更新

中国的行政区划每年都会有变动，如果发现省市区数据未及时更新，欢迎提 Pull Request 帮助我们更新。
