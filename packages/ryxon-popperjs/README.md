# @ryxon/popperjs

@popperjs/core in commonjs format, added Object.assign polyfill.

## Install

```shell
# with npm
npm i @ryxon/popperjs

# with yarn
yarn add @ryxon/popperjs

# with pnpm
pnpm add @ryxon/popperjs
```

## Usage

see: https://popper.js.org/
